---
title: Install
description: "How to install Trane"
---

Trane is available for Python 3.7, 3.8, 3.9, 3.10, and 3.11. It can be installed from PyPI or from source.

# PyPI
To install Trane, run the following command:
```bash
python -m pip install trane
```

# Source
To install Trane from source, clone the repository from Github, and install the dependencies.
```bash
git clone https://github.com/trane-dev/Trane.git
python -m pip install Trane/.
```
